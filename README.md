## 📖 About This Repository

This is the **official companion repository** for:

> **Khan, F.H. (2026).** *Social Science in the Digital Age: Computation, Big Data, AI and the New Research Frontier.* [Publisher].

The book is a **comprehensive graduate-level textbook** spanning **26 chapters across 6 parts** and **6 appendices**, covering the full methodological toolkit of contemporary computational social science (from research design and ethics through machine learning, NLP, network analysis, geospatial methods and generative AI).

This repository contains **every coding exercise, worked example, and dataset** from the book in executable, documented form. All notebooks run in **Google Colab** — no local installation required.

## 🔬 Chapter Highlights & Code Samples

### Chapter 3 — Ethics: The COMPAS Case & Chouldechova Impossibility

One of the book's most powerful pedagogical moments is the mathematical proof that **no algorithm can simultaneously satisfy calibration, FPR parity, and FNR parity when group base rates differ**:

```python
from code.ch03.ethics_audit import FairnessAuditor, chouldechova_impossibility_demo

# Reproduce the ProPublica COMPAS analysis
auditor = FairnessAuditor(
    y_true=recidivism_labels,
    y_pred=compas_scores,
    protected=defendants["race"],
    attr_name="race"
)

# The impossibility: all three metrics cannot hold simultaneously
report = auditor.full_audit_report()
# Output shows: Black defendants FPR = 44.9% vs White = 23.5%
# Yet calibration holds within each group — Chouldechova proven empirically
chouldechova_impossibility_demo()
```

### Chapter 12 — Causal Inference: Difference-in-Differences

Replicating Card & Krueger's minimum wage study — the analysis that overturned the dominant prediction that minimum wages reduce employment:

```python
from code.ch12.causal_inference import DifferenceInDifferences

# Card & Krueger (1994): NJ vs PA fast food employment
# after NJ raised minimum wage from $4.25 to $5.05
did = DifferenceInDifferences(
    data=card_krueger_data,
    outcome="fte_employment",
    treatment_col="new_jersey",
    time_col="post_april_1992",
    unit_col="restaurant_id"
)

result = did.estimate()
# ATT = +2.76 employees (NJ gained, PA lost — against economic prediction)
did.plot_parallel_trends(test_preperiod=True)
did.plot_event_study()
```

### Chapter 16 — NLP: From Word Counts to BERT

```python
from code.ch16.nlp_pipeline import NLPPipeline

# Full pipeline: raw text → sentiment → LDA topics → BERT classification
pipeline = NLPPipeline(language="en")
corpus = pipeline.load_from_csv("data/sample/political_speeches.csv")

# Step 1: Preprocessing
cleaned = pipeline.preprocess(corpus, steps=["lowercase","remove_punct","lemmatise"])

# Step 2: Sentiment analysis (VADER + transformer comparison)
sentiment = pipeline.sentiment_analysis(cleaned, methods=["vader","roberta"])

# Step 3: Topic modelling with LDA (pyLDAvis interactive output)
topics = pipeline.lda_topic_model(cleaned, n_topics=10, visualise=True)

# Step 4: Fine-tuned BERT for political ideology classification
classifier = pipeline.fine_tune_bert(
    train_data=labelled_speeches,
    n_classes=3,        # left / centre / right
    epochs=3
)
```

### Chapter 19 — LLMs as Research Tools: Four Principles

The book's framework for principled AI use (Chapter 19, Section 19.5):

```python
from code.ch19.llm_tools import ResearchLLM, validate_llm_citations

# Principle 1: Full transparency — log every API call
llm = ResearchLLM(
    model="claude-sonnet-4-6",
    log_all_calls=True,         # Required for research disclosure
    temperature=0.0             # Deterministic for reproducibility
)

# Principle 2: Verification — never trust LLM-generated citations
abstract_screen_results = llm.screen_abstracts(
    abstracts=systematic_review_pool,
    inclusion_criteria=PRISMA_criteria
)
validated = validate_llm_citations(abstract_screen_results)
# Flags any hallucinated DOIs or non-existent papers

# Principle 3: Validation — calculate inter-rater reliability with humans
from code.ch19.llm_tools import InterRaterReliability
irr = InterRaterReliability(llm_codes=llm_coding, human_codes=expert_coding)
print(f"Cohen's κ = {irr.kappa():.3f}")   # Must report this in methods
```
